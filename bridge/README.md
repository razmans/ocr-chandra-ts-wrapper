# chandra-ts bridge

FastAPI bridge scaffold for the local Chandra runtime.

Default bind target:
- host: `127.0.0.1`
- port: `8282`

Endpoints planned:
- `GET /health`
- `GET /info`
- `POST /process/file`
- `POST /process/upload`

This scaffold currently returns stubbed responses and is intended to be filled in during implementation.
